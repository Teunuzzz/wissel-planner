# ⚽ Wissel Planner O9 – ZVV De Esch (v2.3.7)

PWA om **wissels** (per 5 min) en **keeperrotatie** eerlijk te verdelen voor O9.  
Werkt op iPhone/Android/laptop en **offline**.

## Highlights
- Eerlijke minuten + optionele **⚖️ Strikte eerlijkheid** (verschil ≤ 1 slot = 5 min).
- **Bench cooldown**: dezelfde speler niet twee slots achter elkaar op de bank (minimaal 2 ertussen).
- **Keeperverdeling**: 1/2/4 (wedstrijd/helften/kwarten), met beleid: kandidaten-eerst / alleen-kandidaten / iedereen.
- **Positie-bewust** (DEF/MID/ATT + allround) en kwaliteitsbalans (drag & drop volgorde).
- **Timer** + **Wissel NU** (tril).
- **Wijzigingen-tab**: keeper kiezen, veld/bank tikken om te wisselen, **Speler stopt**, **Herbereken vanaf nu**.
- **ZVV De Esch** thema.

## Publiceren op GitHub Pages
Kopieer naar je repo-root: `index.html`, `manifest.webmanifest`, `sw.js`, `icon-192.png`, `icon-512.png`, `README.md`.  
Zet Pages aan op `main` (root). Daarna **harde refresh** (cache: `v2_3_7`).
